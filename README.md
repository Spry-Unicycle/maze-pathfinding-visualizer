# Maze Pathfinding Visualizer

An interactive pathfinding visualizer built using C++ and SFML. Users can create custom mazes, place start and end nodes, and watch Breadth-First Search (BFS) explore the grid in real time. The application visualizes visited nodes and reconstructs the shortest path once the destination is reached.

## Features

* Interactive maze editing
* Start and end node placement
* Animated BFS traversal
* Shortest path reconstruction
* Reset functionality
* Real-time graphical visualization using SFML

## Controls

* **W** — Wall placement mode

* **S** — Start node placement mode

* **E** — End node placement mode

* **B** — Run BFS

* **R** — Reset grid

* **Left Click** — Place selected object

* **Right Click** — Erase cell

## Technologies Used

* C++
* SFML 3.x
* Breadth-First Search (BFS)
* Graph Traversal Algorithms
